svgobjects 2.0 Examples Overview

Installation

Eclipse

Mot notably new to this release is the migration of projects to Eclipse. To import the svgobjects distribution into Eclipse:

  1. File -> Import...
  2. Select General -> Existing Projects into Workspace.
  3. Choose the svgobjects2 distribution folder as the root directory.
  4. Check all the projects in the list and click Finish.

Then install each of the frameworks from Eclipse: DirectToiPhone, CoreStore, iPodLook and svgobjects. NOTE: make sure the WebServer resources of DirectToiPhone are installed on a running web server.

Running the Examples

Once the frameworks have been installed, you may build & run the example apps.

This will also require installing the databases per example. You may restore the database from the .bkp file contained in each project. You will however, have to create a Database in Openbase of the same name first.

E.g: To run the iPhone music store example apps, you need to "Restore from ascii" the CoreiTMS.bck located in the folder:

../CoreStore/Database/
